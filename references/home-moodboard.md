# Sense Build Home References

Source images shared inline in chat. File preserves visual direction for impl.

## Reference Notes

- Dark day-planner phone: large blunt number + accent dot; dense/calm type; white rounded bottom sheet from dark upper area.
- Welcome bottom sheet: large friendly heading inside tall rounded white sheet; muted content behind; clear continuation state.
- Notification toasts: pill/sheet components, soft shadows, minimal iconography, semantic color used sparingly.
- Todo app sheet: layered white cards, blurred inactive context, compact floating toolbar, large radius, restrained shadows.
- Custom URL composition: segmented rounded input pieces, oversized type, simple labels, one strong accent color.
- Tribe Capital card: blocky/pixel visual field + editorial text + rounded card geometry.
- Pantom plus motif: static plus-field texture as identity device, not decoration overload.
- Calendar card: soft translucent white surface, rounded top, compact list content, strong hover cursor affordance.

## Home Page Rules

- Keep `Build.` headline with orange stop as primary identity moment.
- Keep plus motif static and close to headline.
- Make prompt sheet the main interactive object.
- Use floating chips + compact controls, not large marketing cards.
- Treat saved apps as soft bottom sheet/list, not dashboard card grid.
- Avoid gradients, heavy color fields, generic AI sparkle effects.
