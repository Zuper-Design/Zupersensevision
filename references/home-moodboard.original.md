# Sense Build Home References

The source images were shared inline in chat, so this file preserves the visual direction locally for implementation.

## Reference Notes

- Dark day-planner phone: large blunt number with an accent dot; dense but calm typography; white rounded bottom sheet emerging from a dark upper area.
- Welcome bottom sheet: large friendly heading inside a tall rounded white sheet; muted content behind; clear continuation state.
- Notification toasts: pill/sheet components with soft shadows, minimal iconography, and semantic color used sparingly.
- Todo app sheet: layered white cards, blurred inactive context, compact floating toolbar, large radius, restrained shadows.
- Custom URL composition: segmented rounded input pieces, oversized type, simple labels, one strong accent color.
- Tribe Capital card: blocky/pixel visual field paired with editorial text and rounded card geometry.
- Pantom plus motif: static plus-field texture used as an identity device, not decoration overload.
- Calendar card: soft translucent white surface, rounded top, compact list content, strong hover cursor affordance.

## Home Page Rules

- Keep the `Build.` headline with the orange stop as the primary identity moment.
- Keep the plus motif static and close to the headline.
- Make the prompt sheet the main interactive object.
- Use floating chips and compact controls rather than large marketing cards.
- Treat saved apps as a soft bottom sheet/list, not a dashboard card grid.
- Avoid gradients, heavy color fields, and generic AI sparkle effects.
